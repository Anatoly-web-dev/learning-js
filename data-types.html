Типы данных в JavaScript

Значение в JavaScript всегда относится к данным определённого типа.

Сущ. 8 основных типов данных в JavaScript

Переменная в JavaScript может содержать любой тип данных.

Числа

Числовой тип данных (number) представляет как целочисленные значения, так и числа с плавающей точкой.

let n = 123; n = 12.345;

Существует множество операций для чисел, например, умножение *, деление /, сложение +, вычитание - и так далее.

Кроме обычных чисел, существуют так называемые «специальные числовые значения», которые относятся к этому типу данных: Infinity, -Infinity, NaN

Infinity представляет собой математическую бесконечность ∞. Это значение, которое больше любого числа, получeнное в результате деления на 0. Можем задать его явно.

NaN означает вычислительную ошибку. NaN - результат неправильной или неопределённой математической операции.

В JavaScript можно использовать букву e, чтобы укоротить запись числа. e добавляется к числу и заменяет указанное количество нулей:

e производит операцию умножения числа на 1 с указанным количеством нулей.

let num = 1e3; // - 1 * 1000 let num = 1.23e6; // - 1.23 * 1000000 отрицательное число после "e"подразумевает деление на 1 с указанным количеством нулей: let num = 1e-3; // - 1 / 1000 (0.001) let num = 1.23e-6; // = 1.23 / 1000000 (0.00000123)

Шестнадцатеричные числа широко используются в JavaScript для представления цветов, кодировки символов и многого другого. Есть короткий стиль записи: 0x, после которого указывается число.

console.log(0xff); // - 255 console.log(0xFF); // - 255 (то же самое, регистр не имеет значения)

Метод num.toString (16) возвращает строковое представление числа для шестнадцатеричного представления.

let num = 255; console.log(num. toString(16)); // ff

Если нужно вызвать метод именно на числе, как toString в примере выше, то нам надо поставить две точки .. после числа.

console.log(225.. toString (16)); // - e1 console.log((225). toString(16)); // - Аналогичный синтаксис Примеры сравнения чисел console.log(0 + -0 === 0 * -0); // - true console.log(3e400 + 5e400 === 8e400); // - true console.log(3e400 + 5e400 === 8e800); // - true console.log(3e40 + 2e40 === 5e40); // - true console.log(3e40 + 2e40 === 5e80); // - false console.log(5e1000 - 5e1000 == NaN); // - false console.log(5e1000 - 5e1000 == 0); // - false console.log(5e1000 - 5e1000 == -0); // - false

Округление

При работе с дробями происходит потеря точности вычеслений.

В JavaScript есть несколько встроенных функций для работы с округлением. Эти функции охватывают все возможные способы обработки десятичной части.

Math.floor - округление в меньшую сторону: 3.1 становится 3, а -1.1 — -2.

Math.ceil - округление в большую сторону: 3.1 становится 4, а -1.1 — -1.

Math.round - округление до ближайшего целого: 3.1 становится 3, 3.6 — 4, а -1.1 — -1.

Math.trunc - удаление дробной части без округления: 3.1 становится 3, а -1.1 — -1.

let num = 1.23456; console.log(Math. floor(num)); // - 1 console.log(Math. round(12.57)); // - 13

Метод toFixed(n) округляет число до n знаков после запятой и возвращает строковое представление результата.

console.log(1.33333. toFixed(2)); // - 1.33 console.log(1.67333. toFixed(1)); // - 1.7

isNaN и isFinite

isNaN(значение) преобразует значение в число и проверяет является ли оно NaN:

console.log(isNaN (NaN)); // - true console.log(isNaN ("str")); // - true

isFinite(значение) преобразует аргумент в число и возвращает true, если оно является обычным числом, т.е. не NaN / Infinity / -Infinity.

console.log(isFinite("15")); // - true console.log(isFinite(false)); // - true console.log(isFinite('')); // - true

parseInt и parseFloat

Чтобы получить числовое значение из строк, содержащих числовые и строчные символы ("100px", "19€"и т.д.) существуют функции parseInt и parseFloat. Они «читают» число из строки. Если в процессе чтения возникает ошибка, они возвращают полученное до ошибки число.

parseInt возвращает целое число.

console.log(parseInt('300px')); // - 300

parseFloat возвращает число с плавающей точкой.

console.log(parseFloat('12.5px')); // - 12.5

Функции parseInt/parseFloat вернут NaN, если не смогли прочитать ни одну цифру и если строка начинается с буквы.

console.log(parseInt('a123')); // - NaN

Функция parseInt имеет необязательный 2ой параметр. Он определяет систему счисления, т.о. parseInt может читать строки с 16-ричными числами

console.log(parseInt('0xff', 16)); // - 255

Другие математические функции

В JavaScript встроен объект Math, содержащий различные математические функции и константы, включая тригонометрические функции.

Math.random() - возвращает псевдослучайное число в диапазоне от 0 (включительно) до 1 (но не включая 1).

Math.max (число-1, ... число-n) - возвращает наибольшее число из перечисленных аргументов.

Math.min (число-1, ... число-n) - возвращает наименьшее число из перечисленных аргументов.

Math.pow (n, power) - возвращает число-n, возведённое в степень-power.

BigInt

В JavaScript тип «number» не может содержать числа больше, чем (253-1) (т.е. 9007199254740991), или меньше, чем -(253-1) для отрицательных чисел. Это техническое ограничение.

Иногда нам нужны действительно гигантские числа, например, в криптографии или при использовании метки времени («timestamp») с микросекундами.

Тип BigInt был добавлен в JavaScript, чтобы дать возможность работать с целыми числами произвольной длины.

Чтобы создать значение типа BigInt, необходимо добавить n в конец числового литерала:

const bigInt = 1234567890123456789012345678901234567890n;

Строки

Строка (string) в JavaScript должна быть заключена в кавычки.

В JavaScript существует три типа кавычек.

Двойные или одинарные кавычки являются «простыми», между ними нет разницы. Например: "строка", 'строка'.

Обратные кавычки позволяют встраивать переменные, заключая их в ${…}. Например: `строка ${ переменная }`.

let name = "Иван"; alert(`Привет, ${name}!`); // - получим строку: Привет, Иван! alert(`результат: ${1 + 2}`); // - получим строку: результат: 3 // - Для вывода в консоль в таком виде используем также обратные кавычки `` console.log (`Список покупок: 1. Молоко; 2. Яблоки; 3. Курица;`);

Выражение внутри ${…} вычисляется, и его результат становится частью строки. Мы можем положить туда всё, что угодно: переменную или выражение 1 + 2, или что-то более сложное.

Строка может содержать ноль символов (быть пустой ''), один символ или множество символов.

Методы для работы со строками

console.log ('Магазин \ 'У дома \''); // - Для добавления кавычек используем обратный слэш \ Если нам надо написать в строке \, надо использовать двойной обратный слэш \\ (получим в результате один слэш \

Cвойство length показывает количество символов в строке.

console.log('Привет!' .length); // - получим 7

Указав в [] порядковый номер после строки, мы получим определенный символ строки. Нумерация начинается с 0.

console.log('hello world' [1] ); // - получим второй символ строки - e

Последний символ строки мы можем получить, отняв от длины строки 1. Например: str[ str. length - 1]).

Используя метод toUpperCase() и квадратные скобки [] мы можем перевести определенный символ строки в верний регистр. Например:

Метод toLowerCase() используют для перевода строки в нижний регистр.

Метод toUpperCase() используют для перевода строки в верхний регистр.

let str = 'hello world'; str = str [0] .toUpperCase() + str.slice(1); console.log(str); // - получим Hello world

Метод includes() проверяет наличие в строке искомой подстроки. Eсли искомая подстрока есть, то мы получим true, иначе false. Второй необязательный аргумент указывает позицию, с которой следует начать поиск.

let str = 'Hello world'; console.log( str.includes ('world')); // - получим true;

Метод slice(begin, end) возвращает новую строку начиная с begin до end (не включительно). Если передали только один аргумент begin, то вернется подстрока начиная с begin и до конца строки.

Также можно задать позицию с конца строки, указав отрицательные индексы у аргументов.

let str = 'hello world!'; str = str.slice (6, -2); conslole.log(str); // - получим world

Метод repeat(count) продублирует строку count раз.

let str = 'hello_'; str = str.repeat (3); conslole.log(str); // - получим hello_hello_hello

Метод trim() убирает лишние пробелы у строки.

Булевый(Логический) тип

Булев тип назначается автоматически при присваивании.

Не стоит воспринимать булевы значения как значения стопроцентной истины и лжи. Булевы значения могут быть интерпретированы, например, как:

  • Включен/выключен модуль.
  • Имеет права администратора/не имеет прав.
  • Отмечена галочка «отправлять письма»/не отмечена.

Присваивать булевым переменным значения можно в результате вычисления выражений и от других значений и функций.

Булевый тип (boolean) может принимать только два значения: true (истина) и false (ложь).

Булевый тип используется для хранения значений да / нет: true значит «да, а false значит «нет. Булевые значения также могут быть результатом сравнений:

let isGreater = 4 > 1; alert(isGreater); // - true (результатом сравнения будет "да")

Значения null и undefined

Обычно null используется для присвоения переменной «пустого» или «неизвестного» значения, а undefined – для проверок, была ли переменная назначена. Если переменная объявлена, но ей не присвоено никакого значения, то её значением будет undefined.

let age; alert(age); // - выведет undefined

Объекты и символы

Тип object (объект) – особенный. Все остальные типы называются «примитивными», потому что их значениями могут быть только простые значения (будь то строка, или число, или что-то ещё). В объектах же хранят коллекции данных или более сложные структуры. Тип symbol (символ) используется для создания уникальных идентификаторов в объектах.

Преобразование типов

Чаще всего операторы и функции приводят автоматически переданные им значения к нужному типу.

Например, alert автоматически преобразует любое значение к строке. Математические операторы преобразуют значения к числам. Есть также случаи, когда нам нужно явно преобразовать значение в ожидаемый тип.

Оператор typeof возвращает тип аргумента. Это полезно, когда мы хотим обрабатывать значения различных типов по-разному или просто хотим сделать проверку.

Оператор typeof позволяет увидеть, какой тип данных сохранён в переменной. Возвращает строку с именем типа. Например, "string". Для null возвращается "object" – это ошибка в языке, на самом деле это не объект.

Строковое преобразование

Строковое преобразование происходит, когда требуется представление чего-либо в виде строки.

Например, alert( value) преобразует value к строке.

Также мы можем использовать функцию String(value), чтобы преобразовать value к строке:

let value = true; alert(typeof value); // - выведет boolean. value = String( value); // - теперь value это строка "true" alert(typeof value); // - выведет string

Преобразование происходит очевидным образом. false становится "false", null становится "null" и т.п.

Численное преобразование

Численное преобразование происходит в математических функциях и выражениях.

Например, когда операция деления / применяется не к числу:

alert( "6" / "2" ); // - 3, строки преобразуются в числа

Мы можем использовать функцию Number(значение), чтобы явно преобразовать значение к числу:

let str = "123"; alert(typeof str); // - string let num = Number(str); // - становится числом 123 alert(typeof num); // - результат: number

Явное преобразование применяется, когда ожидаем получить число из строки, например из текстовых полей форм.

Если строка не может быть явно приведена к числу, то результатом преобразования будет NaN. Например:

let age = Number("Любая строка"); alert(age); // - NaN, преобразование не удалось

Правила численного преобразования:

  • undefined преобразуется в NaN
  • null преобразуется в 0
  • true преобразуется в 1
  • false преобразуется в 0
Примеры: alert(Number( " 123") ); // - 123 alert(Number( "123z")); // - NaN (ошибка чтения числа на месте символа "z") alert(Number( true)); // - 1 alert(Number( false)); // - 0

Логическое преобразование

Логическое преобразование происходит в логических операциях, но также может быть выполнено явно с помощью функции Boolean ( значение ).

Значения, которые интуитивно «пустые>», вроде 0, пустой строки, null, undefined и NaN, становятся false. Все остальные значения становятся true.

alert( Boolean (1) ); // true alert( Boolean (0) ); // false alert( Boolean ("Привет!") ); // true alert( Boolean ("") ); // false alert( Boolean ("0") ); // true alert( Boolean (" ") ); // пробел это тоже true
