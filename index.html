Введение в JavaScript

Язык программирования JavaScript придумали специально для того, чтобы создавать интерактивные сайты.

В браузере JavaScript может:

  • Добавлять новый HTML-код на страницу, изменять существующее содержимое, модифицировать стили;
  • Реагировать на действия пользователя, щелчки мыши, перемещения указателя, нажатия клавиш;
  • Отправлять сетевые запросы на удалённые сервера, скачивать и загружать файлы (технологии AJAX и COMET);
  • Получать и устанавливать куки, задавать вопросы посетителю, показывать сообщения
  • Запоминать данные на стороне клиента («local storage»)

JavaScript позволяет делать приложения не только в браузерах, но и на сервере, на мобильных устройствах и т.п.

Справочники по JavaScript:

  • MDN (Mozilla) JavaScript Reference
  • MSDN – справочник от Microsoft

Код на языке JavaScript называют скриптом. Его сохраняют в отдельный файл с расширением js, а чтобы запустить, подключают этот файл на страницу. В HTML для добавления JavaScript есть специальный тег:

script src = "адрес_файла">/script

Подключают скрипт обычно в самом конце страницы, перед закрывающим тегом /body.

Программы на JavaScript также могут быть вставлены в любое место HTML-документа с помощью тега script.

Программа на JavaScript — это последовательность инструкций, то есть указаний браузеру выполнить какие-то действия. Инструкции выполняются последовательно, сверху вниз.

Инструкции в JavaScript могут состоять из значений, операторов, выражений, ключевых слов, комментариев.

Чтобы сказать JavaScript, что инструкция закончена, нужно поставить (;) или перейти на новую строку. Новая строка правильно работает в большинстве случаев, а (;) — всегда. Поэтому лучше ставить (;) в конце каждой инструкции.

JavaScript не меняет исходный файл с разметкой, но, выполняя инструкции, меняет страницу прямо в браузере пользователя.

Значения, операторы, выражения, ключевые слова

В JavaScript есть несколько типов значений (values): фиксированные значения (литералы), значения констант и значения переменных.

В JavaScript есть много различных операторов, они обозначают операцию, которую нужно произвести. Например: * , / , + , - , = и т.д.

Выражения - это комбинация значений, переменных и операторов, которые либо присваивают переменной значениe, либо возвращают какое либо значениe без его присваивания.

let num = 4; // - выражение присвоения

console.log (num);

'Учим' + 'JS'; // - выражение без присвоения

Инструкции JavaScript часто начинаются с ключевого слова, определяющего какое действие нужно выполнить JavaScript.

let - ключевое слово для создания (объявления) переменной.

const - ключевое слово для объявления константы.

Блок инструкций

JavaScript инструкции можно сгруппировать с помощью блока, по сути фигурных скобок {....}. Как правило используется в ' функциях, циклах и т.д.

function name() { console.log('Учим'); console.log('JS'); }

Область видимости переменных

Комментарии

Комментарий — это текст, поясняющий код. Он не выводится в браузер и никак не влияет на работу программы. Инструкции внутри комментария не выполняются, поэтому комментарии часто используют, если нужно временно отключить часть кода.

В JavaScript есть два вида комментариев:

// - Однострочные комментарии. /* - Многострочные комментарии. Они могут отключить сразу несколько строк кода. */

Инструменты разработчика

Инструменты разработчика встроенные в браузере (F12) позволяют нам смотреть ошибки, выполнять команды, проверять значение переменных и ещё много всего полезного.

Консоль — инструмент разработчика, который помогает тестировать код. Если во время выполнения скрипта возникнет ошибка, в консоли появится сообщение о ней. А ещё в консоль можно выводить текстовые подсказки. Чтобы вывести сообщение в консоль, нужно использовать console.log( ):

console.log('Hello, world!'); // - Выведет: 'Hello, world!' console.log (document. querySelector ('.page')); // - Выведет в консоль найденный элемент.

Модальные окна глобального объект window

alert

alert( ) - показывает сообщение и ждёт, пока пользователь нажмёт кнопку «ОК». Это небольшое окно с сообщением называется модальным окном.

Понятие модальное означает, что пользователь не может взаимодействовать с интерфейсом остальной части страницы, до тех пор, пока взаимодействует с окном. В данном случае – пока не будет нажата кнопка «OK».

alert('Hello');

prompt

Функция prompt( ) вызывает модальное окно с текстом и полем для ввода текста и кнопками OK / Отмена.prompt( ) принимает два аргумента:

В примере ниже Пользователь может напечатать что-либо в поле ввода и нажать кнопку OK. Введённый текст будет присвоен переменной result. Пользователь также может отменить ввод нажатием на кнопку «Отмена» или нажав на клавишу Esc. В этом случае значением переменной result станет null. Вызов prompt( ) возвращает строковое значение, указанное в поле ввода, или null, если ввод отменён пользователем. let result = prompt('title', 'default'); title - Текст для отображения в окне. default - необязательный второй параметр, который устанавливает начальное значение в поле для текста в окне. Квадратные скобки [] вокруг 'default' в описанном выше синтаксисе означают, что параметр необязательный. let age = prompt('Сколько тебе лет?', 10); alert(`Тебе ${age} лет!`); // - вернет строку 'Тебе 10 лет!'

confirm

Функция confirm( ) отображает модальное окно с текстом вопроса и двумя кнопками: OK / Отмена.

Результат – true, если нажата кнопка OK. В других случаях – false.

let isBoss = confirm('Ты здесь главный?'); alert(isBoss); // - вернёт true, если нажата кнопка OK

Все эти методы являются модальными: останавливают выполнение скриптов и не позволяют пользователю взаимодействовать с остальной частью страницы до тех пор, пока окно не будет закрыто.

Расположение окон определяется браузером. Обычно окна находятся в центре. Визуальное отображение окон зависит от браузера, и мы не можем изменить их вид.

Есть другие способы показать более приятные глазу окна с богатой функциональностью для взаимодействия с пользователем.

Переменные и константы

Переменная — способ сохранить данные, дав им понятное название. Переменная определяется как имя, с которым может быть связано значение.

Переменную можно создать(объявить), с помощью ключевого слова let. За ним следует имя переменной. После объявления в переменную нужно записать (присвоить) какое-то значение:

let header = document.querySelector ('header');

Имя переменной может быть почти любым, но не должно начинаться с цифры, а из спецсимволов разрешены только '_' и '$'. Ещё нельзя использовать зарезервированные слова.

Имена переменных чувствительны к регистру: header, Header и HEADER — это разные переменные.

Имя переменной должно описывать то, что в ней хранится.

Если имя переменной состоит из нескольких слов, то принято записывать его в стиле lowerCamelCase, например: userName.

Когда в коде встречается переменная, браузер вместо её имени подставляет присвоенное ей значение. Когда мы используем переменную, снова писать let не нужно:

console.log(header);

Чтобы объявить несколько переменных можно записать их одним из способов:

  • через запятую в одну строку let myName = 'Анатолий', myAge = '30';
  • через запятую с новой строки let myName = 'Анатолий', myAge = '30';
  • отдельно let myName = 'Анатолий'; let myAge = '30';

Передача значений переменных:

let myAge = 30; let myNewAge = 40; myAge = myNewAge; console.log(myAge); // - 40

Область видимости переменных

Объявление переменных в разных частях JavaScript кода имеет значение. Это область видимости переменных.

Видимость переменной только внутри блока инструкций function testBlock() { let myAge = 36; console.log(myAge); } console.log(myAge); // - переменная не видна за пределами блока. Переменные внутри и за пределами блока let myAge = 30; function testBlock() { let myAge = 40; console.log(myAge); // - выведет в консоли значение 40 } console.log(myAge); // - выведет в консоли значение 30, которое находится вне блока.

Мы не можем использовать переменную перед ее объявлением.

Данная запись вызовет ошибку: console.log(myAge); let myAge = 36;

Повторное объявление переменной в рамках одного блока запрещено. В тоже время, внутри блока (например цикла) проблем нет.

Константа - это таже переменная, но которую нельзя изменить

Данная запись вызовет ошибку, нельзя изменить константу. const myAge = 30; myAge = 31;

Во всем остальном, что касается правил имен констант, области видимости отличий с переменными нет.

Имена констант с заранее известными значениями принято записывать в ВЕРХНЕМ РЕГИСТРЕ, а слова разделять (_ )

const BLOCK_SIZE = 25; const COLOR_RED = "#F00"; Имя константы со значениями присвоенными в процессе выполнения программы: const summSizes = 25 + 30;
